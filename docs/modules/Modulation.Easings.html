Namespace Easings

Easings module

See the guide

Overview:

  • Easings.create: Create an easing with provided settings
  • Easings.time: Ease by time
  • Easings.ticks: Ease by tick
  • Easings.get: Get an easing function by name

Example: Importing

// If library is stored two directories up under `ixfx/`
import { Easings } from '../../ixfx/dist/modulation.js';
Easings.time(...);

// Import from web
import { Easings } from 'https://unpkg.com/ixfx/dist/modulation.js'
Easings.time(...);
