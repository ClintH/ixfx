Easing

Easing: HasCompletion & { get isDone(): boolean; compute: any; reset: any }

'Ease' from 0 to 1 over a delicious curve. Commonly used for animation and basic modelling of phyical motion.

Create via tick or time, call compute to calculate the next value in the progression, until you reach 1 or isDone returns true.

crossfade

  • crossfade(amt: number, easingA: EasingFn, easingB: EasingFn): number

  • Returns a 'crossfade' of two easing functions, synchronised with the progress through the easing. That is:

    • 0.0 will yield 100% of easingA at its easing(0) value.
    • 0.2 will yield 80% of a, 20% of b, with both at their easing(0.2) values
    • 0.5 will yield 50% of both functions both at their easing(0.5) values
    • 0.8 will yield 20% of a, 80% of a, with both at their easing(0.8) values
    • 1.0 will yield 100% of easingB at its easing(1) value.

    So easingB will only ever kick in at higher amt values and easingA will only be present in lower valus.

    Parameters

    • amt: number
    • easingA: EasingFn
    • easingB: EasingFn

    Returns number

    Numeric value

fromCubicBezier

  • fromCubicBezier(b: number, d: number): EasingFn

  • Creates an easing function using a simple cubic bezier defined by two points.

    Eg: https://cubic-bezier.com/#0,1.33,1,-1.25 a:0, b: 1.33, c: 1, d: -1.25

    // Time-based easing using bezier
    const e = Easings.time(fromCubicBezier(1.33, -1.25), 1000);
    e.compute();

    Parameters

    • b: number
    • d: number

    Returns EasingFn

    Value

mix

  • mix(amt: number, balance: number, easingA: EasingFn, easingB: EasingFn): number

  • Returns a mix of two easing functions.

    // Get a 50/50 mix of two easing functions at t=0.25
    mix(0.5, 0.25, sineIn, sineOut);

    // 10% of sineIn, 90% of sineOut
    mix(0.90, 0.25, sineIn, sineOut);

    Parameters

    • amt: number

      'Progress' value passed to the easing functions

    • balance: number

      Mix between a and b

    • easingA: EasingFn
    • easingB: EasingFn

    Returns number
