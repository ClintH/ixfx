Easing

Easing: HasCompletion & { get isDone(): boolean; compute: any; reset: any }

'Ease' from 0 to 1 over a delicious curve. Used commonly for animation and basic modelling of phyical motion.

Create via easeOverTicks or easeOverTime, call compute to calculate the next value in the progression, until you reach 1 or isDone returns true.
