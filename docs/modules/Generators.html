Namespace Generators

Generators produce values on demand.

Overview:

  • count: Generate a set numbers, counting by one
  • numericPercent: Generate a range of numbers on the percentage scale of 0-1
  • numericRange: Generate a range of numbers
  • pingPong / pingPongPercent: Generate numbers that repeat up and down between the set limits

Aliases:

  • delayLoop: A generator that yields at a given rate
  • interval: Generates values from a given function with a given delay
  • randomUniqueInteger: Random unique integer

Example

Importing

// If library is stored two directories up under `ixfx/`
import {count, interval} from '../../ixfx/dist/generators.js';
// Import from web
import {count, interval} from 'https://unpkg.com/ixfx/dist/generators.js'
