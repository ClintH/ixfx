Namespace Bipolar

Work with bipolar values (-1...1)

Import:

import { Bipolar } from 'https://unpkg.com/ixfx/dist/data.js';

Overview:

  • clamp: Clamp on -1..1 scale
  • fromScalar: Convert from 0..1 to -1..1
  • immutable: Immutable wrapper around a value
  • random: Create a random bipolar value
  • scale: Scale a value to -1..1 (clamped)
  • scaleUnclamped As scale but allowed to go outside of -1...1 range
  • toScalar: Convert -1..1 to 0..1
  • towardZero: Nudge a bipolar value towards zero
