Palette

Palette: { add: any; get: any; getOrAdd: any; has: any; setElementBase: any }

Manage a set of colours. Uses CSS variables as a fallback if colour is not added

  • get:function
    • get(key: string, fallback?: string): string

    • Returns a colour by name.

Const getCssVariable

  • getCssVariable(name: string, fallbackColour?: string, root?: HTMLElement): string

  • Gets a CSS variable.

    example

    Fetch --accent variable, or use yellow if not found.

    getCssVariable(`accent`, `yellow`);

    Parameters

    • name: string

      Name of variable. Do not starting --

    • fallbackColour: string = ...

      Fallback colour if not found

    • Optional root: HTMLElement

      Element to search variable from

    Returns string

    Colour or fallback.
