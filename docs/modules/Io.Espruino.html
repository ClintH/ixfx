EvalOpts

EvalOpts: { assumeExclusive?: boolean; timeoutMs?: number }

Options for code evaluation

Type declaration

  • Optional Readonly assumeExclusive?: boolean

    If true (default), it assumes that anything received from the board is a response to the eval

  • Optional Readonly timeoutMs?: number

    Milliseconds to wait before giving up on well-formed reply. 5 seconds is the default.

Options

Options: { debug?: boolean; evalTimeoutMs?: number; name?: string }

Options for device

Type declaration

  • Optional Readonly debug?: boolean

    If true, additional logging information is printed

  • Optional Readonly evalTimeoutMs?: number

    Default milliseconds to wait before giving up on a well-formed reply. 5 seconds is the default.

  • Optional Readonly name?: string

    Name of device. Only used for printing log mesages to the console
