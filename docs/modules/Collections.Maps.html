Namespace Maps

ixfx also includes MapMutable, MapImmutable

Const addUniqueByHash

  • addUniqueByHash<V>(set: undefined | ReadonlyMap<string, V>, hashFunc: ToString<V>, ...values: readonly V[]): Map<any, any>

  • Adds items to a map only if their key doesn't already exist

    Uses provided ToString function to create keys for items. Item is only added if it doesn't already exist. Thus the older item wins out, versus normal Map.set where the newest wins.

    example
    const map = new Map();
    const peopleArray = [ _some people objects..._];
    addUniqueByHash(map, p => p.name, ...peopleArray);

    Type parameters

    • V

    Parameters

    • set: undefined | ReadonlyMap<string, V>
    • hashFunc: ToString<V>
    • Rest ...values: readonly V[]

    Returns Map<any, any>

Const hasAnyValue

  • hasAnyValue<K, V>(map: ReadonlyMap<K, V>, value: V, comparer: IsEqual<V>): boolean

  • Returns true if any key contains value, based on the provided comparer function. Use hasKeyValue if you only want to find a value under a certain key.

    Having a comparer function is useful to check by value rather than object reference.

    example

    Finds value samantha, using string equality to match

    hasAnyValue(map, `samantha`, (a, b) => a === b);

    Type parameters

    • K

    • V

    Parameters

    • map: ReadonlyMap<K, V>

      Map to search

    • value: V

      Value to find

    • comparer: IsEqual<V>

      Function that determines matching

    Returns boolean

    True if value is found

Const mapToArray

  • mapToArray<K, V, R>(m: ReadonlyMap<K, V>, transformer: (key: K, item: V) => R): readonly R[]

  • Converts Map<K,V> to Array with a provided transformer

    example

    Get a list of ages from a map of Person objects

    let person = { age: 29, name: `John`};
    map.add(person.name, person);
    const ages = mapToArray<string, People, number>(map, (key, person) => person.age);
    // [29, ...]
