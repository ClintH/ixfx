Namespace Triangles

Helpers for creating:

  • Triangles.fromFlatArray: Create from [x1, y1, x2, y2, x3, y3]
  • Triangles.fromPoints: Create from three {x,y} sets
  • Triangles.fromRadius: Equilateral triangle of a given radius and center

apply

  • apply(t: Triangle, fn: (p: Point, label?: string) => Point): Readonly<Triangle>

  • Applies fn to each of a triangle's corner points, returning the result.

    example

    Add some random to the x of each corner

    const t = apply(tri, p => {
     const r = 10;
     return {
       x: p.x + (Math.random()*r*2) - r,
       y: p.y
     }
    });

    Parameters

    • t: Triangle
    • fn: (p: Point, label?: string) => Point
        • (p: Point, label?: string): Point

        • Parameters

          • p: Point
          • Optional label: string

          Returns Point

    Returns Readonly<Triangle>

fromRadius

  • fromRadius(origin: Point, radius: number, opts?: { initialAngleRadian?: number }): Triangle

  • Returns an equilateral triangle centered at the origin.

    // Create a triangle at 100,100 with radius of 60
    const tri = equilateralFromOrigin({x:100,y:100}, 60);

    // Triangle with point A upwards, B to the right, C to the left
    constr tri2 = equilateralFromOrigin({x:100,y:100}, 60, {initialAngleRadian: -Math.PI / 2});

    Parameters

    • origin: Point
    • radius: number
    • opts: { initialAngleRadian?: number } = {}

    Returns Triangle

rotate

  • rotate(t: Triangle, amountRadian?: number, origin?: Point): Triangle

  • Returns a triangle that is rotated by angleRad. By default it rotates around its center but an arbitrary origin point can be provided.

    // Rotate triangle by 5 degrees
    rotate(triangle, degreeToRadian(5));

    // Rotate by 90 degrees
    rotate(triangle, Math.PI / 2);

    Parameters

    • t: Triangle
    • Optional amountRadian: number

      Angle in radians to rotate by

    • Optional origin: Point

      Point to rotate around. If undefined, middle of line will be used

    Returns Triangle
