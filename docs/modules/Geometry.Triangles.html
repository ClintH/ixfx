Namespace Triangles

Triangle processing.

Helpers for creating:

  • Triangles.fromFlatArray: Create from [ x1, y1, x2, y2, x3, y3 ]
  • Triangles.fromPoints: Create from three { x, y } sets
  • Triangles.fromRadius: Equilateral triangle of a given radius and center

There are sub-modules for dealing with particular triangles:

  • Triangles.Equilateral: Equilateral triangls
  • Triangles.Right: Right-angled triangles
  • Triangles.Isosceles: Iscosceles triangles

Calculations

  • angles: Internal angles in radians. anglesDegrees for degrees.
  • area: Area of triangle
  • bbox: Bounding box
  • centroid: Centroid of triangle
  • perimeter: Calculate perimeter
  • lengths: Return array lengths of triangle's edges
  • rotate, rotateByVertex

Conversions

  • edges: Edges of triangle as Line
  • corners: Corner positions
  • innerCircle: Largest circle to fit within triangle
  • outerCircle: Largest circle to enclose triangle
  • toFlatArray

Comparisons

  • intersectsPoint: Whether a point intersects triangle
  • isEqual: Check whether two triangles have equal values
  • isAcute, isEquilateral, isIsosceles, isRightAngle
  • isTriangle: Returns true if object has expected properties of a triangle
