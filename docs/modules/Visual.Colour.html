Namespace Colour

Colour interpolation, scale generation and parsing

Overview

  • interpolator: Blend colours
  • scale: Produce colour scale
  • opacity: Give a colour opacity
  • randomHue: Generate a random hue
  • goldenAngleColour: Pick perceptually different shades

CSS

  • getCssVariable: Parse a CSS-defined colour

Conversions: convert from 'blue', 'rgb(255,0,0)', 'hsl(0, 100%, 50%)' etc:

  • toHex: to a hex format string
  • toHsl: to a {h, s, l} object
  • toRgb: to a {r, g, b} object
