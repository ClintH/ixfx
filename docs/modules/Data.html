Namespace Data

Processing streams of data. Read more in the docs

Sub-modules:

  • Data.Arrays helpers for working with arrays
  • Data.Correlate simple data correlation
  • Data.Maps helpers for working with maps
  • Data.Pathed: compare objects, access value by string path
  • Data.Pool: allocate resources within a pool
  • Data.Process: process values

Comparing

  • Data.changedDataFields: set of fields that have changed between two objects
  • Data.compareData: produce a list of changes between two objects
  • Data.compareArrays: compare contents of an array
  • Data.compareKeys: compare keys of two objects

Etc

  • keysToNumbers: Convert keys of an object to number data type
  • mapObjectByObject: Map values of an object using a structured set of map functions. Allows you to use different map functions for different keys.
  • mapObjectShallow: Maps top-level values of an object through a common function.
  • resolve: For an input value, iterator/generator or function, it will return a value
  • resolveFields: Like resolve but works on all the top-level values of an object.

Example: Importing

// If library is stored two directories up under `ixfx/`
import { resolve } from '../../ixfx/dist/data.js';
// Import from web
import { resolve } from 'https://unpkg.com/ixfx/dist/data.js'
