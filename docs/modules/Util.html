getFieldByPath

  • getFieldByPath(o: any, path?: string): any

  • Returns a field on object o by a dotted path.

    const d = {
     accel: {x: 1, y: 2, z: 3},
     gyro:  {x: 4, y: 5, z: 6}
    };
    getFieldByPath(d, `accel.x`); // 1
    getFieldByPath(d, `gyro.z`);  // 6
    getFieldByPath(d, `gyro`);    // {x:4, y:5, z:6}
    getFieldByPath(d, ``);        // Returns original object

    If a field does not exist, undefined is returned. Use getFieldPaths to get a list of paths.

    Parameters

    • o: any
    • path: string = ...

    Returns any

getFieldPaths

  • getFieldPaths(o: any): readonly string[]

  • Returns a list of paths for all the fields on o

    const d = {
     accel: {x: 1, y: 2, z: 3},
     gyro:  {x: 4, y: 5, z: 6}
    };
    const paths = getFieldPaths(d); 
    // Yields [ `accel.x`, `accel.y`,`accel.z`,`gyro.x`,`gyro.y`,`gyro.z` ]

    Use getFieldByPath to fetch data by this 'path' string.

    Parameters

    • o: any

    Returns readonly string[]
