average

  • average(...numbers: readonly number[]): number

  • Calculates the average of all numbers in an array. Array items which aren't a valid number are ignored and do not factor into averaging.

    example
    import * as Numbers from 'https://unpkg.com/ixfx/dist/numbers.js';

    // Average of a list
    const avg = Numbers.average(1, 1.4, 0.9, 0.1);

    // Average of a variable
    let data = [100,200];
    Numbers.average(...data);

    See also: Arrays.average which takes an array.

    Parameters

    • Rest ...numbers: readonly number[]

    Returns number

filter

  • filter(it: Iterable<unknown>): Generator<unknown, void, unknown>

  • Filters an iterator of values, only yielding those that are valid numbers

    import * as Numbers from 'https://unpkg.com/ixfx/dist/numbers.js';

    const data = [true, 10, '5', { x: 5 }];
    for (const n of Numbers.filter(data)) {
     // 5
    }

max

  • max(...data: readonly number[]): number
