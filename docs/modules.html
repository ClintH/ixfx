Const clamp

  • clamp(v: number, min?: number, max?: number): number

  • Clamps a value between min and max (both inclusive) Defaults to a 0-1 range, useful for percentages.

    Usage: clamp(0.5); // 0.5 - just fine, within default of 0 to 1 clamp(1.5); // 1 - above default max of 1 clamp(-50, 0, 100); // 0 - below range clamp(50, 0, 50); // 50 - within range

    For clamping integer ranges, consider clampZeroBounds

    Parameters

    • v: number

      Value to clamp

    • min: number = 0
    • max: number = 1

    Returns number

    Clamped value

Const clampZeroBounds

  • clampZeroBounds(v: number, length: number): number

  • Clamps integer v between 0 (inclusive) and length (exclusive) This is useful for clamping an array range, because the largest allowed number will be one less than length

    const myArray = [`a`, `b`, `c`, `d`];
    clampZeroBounds(0, myArray.length); // 0
    clampZeroBounds(1.2, myArray.length); // 1
    clampZeroBounds(4, myArray.length); // 4
    clampZeroBounds(5, myArray.length); // 4
    clampZeroBounds(-1, myArray.length); // 0

    Parameters

    • v: number

      Integer value to clamp

    • length: number

      Length of bounds

    Returns number

    Clamped value, minimum will be 0, maximum will be one less than length.
