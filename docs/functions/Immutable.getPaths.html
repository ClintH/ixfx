Function getPaths

  • getPaths(object, onlyLeaves?): readonly string[]

  • Returns a list of paths for all the fields on o

    const d = {
     accel: { x: 1, y: 2, z: 3 },
      gyro: { x: 4, y: 5, z: 6 }
    };
    const paths = getFieldPaths(d);
    // Yields [ `accel`, `gyro`, `accel.x`, `accel.y`,`accel.z`,`gyro.x`,`gyro.y`,`gyro.z` ]

    Use getField to fetch data by this 'path' string.

    If object is null, an empty array is returned.

    If onlyLeaves is true, only leaf nodes are included. false by default.

    const paths = getFieldPaths(d, true);
    // Yields [ `accel.x`, `accel.y`,`accel.z`,`gyro.x`,`gyro.y`,`gyro.z` ]

    Parameters

    • object: null | object

      Object to get paths for

    • onlyLeaves: boolean = false

      If true, only paths with a primitive value are returned.

    Returns readonly string[]
