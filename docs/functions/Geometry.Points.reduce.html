Function reduce

  • reduce(pts, fn, initial?): Point

  • Reduces over points, treating x and y separately.

    // Sum x and y values
    const total = Points.reduce(points, (p, acc) => {
     return {x: p.x + acc.x, y: p.y + acc.y}
    });

    Parameters

    • pts: readonly Point[]

      Points to reduce

    • fn: ((p, accumulated) => Point)

      Reducer

        • (p, accumulated): Point

        • Parameters

          • p: Point
          • accumulated: Point

          Returns Point

    • initial: Point = ...

      Initial value, uses { x:0, y:0 } by default

    Returns Point
