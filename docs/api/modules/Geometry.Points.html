Const findMinimum

  • findMinimum(compareFn: (a: Point, b: Point) => Point, ...points: readonly Point[]): Point

  • Returns the 'minimum' point from an array of points, using a comparison function.

    example

    Find point closest to a coordinate

    const points = [...];
    const center = {x: 100, y: 100};

    const closestToCenter = findMinimum((a, b) => {
     const aDist = distance(a, center);
     const bDist = distance(b, center);
     if (aDistance < bDistance) return a;
     return b;
    }, points);

    Parameters

    • compareFn: (a: Point, b: Point) => Point

      Compare function returns the smallest of a or b

          Returns Point

    Returns Point

Const from

  • from(xOrArray?: number | readonly number[], y?: number): Point

  • Returns a point from two coordinates or an array of [x,y]

    example
    let p = from([10, 5]); // yields {x:10, y:5}
    let p = from(10, 5);   // yields {x:10, y:5}
    let p = from(10);      // yields {x:10, y:0} 0 is used for default y
    let p = from();        // yields {x:0, y:0}  0 used for default x & y

    Parameters

    • Optional xOrArray: number | readonly number[]
    • Optional y: number

    Returns Point

Const withinRange

  • withinRange(a: Point, b: Point, maxRange: number | Point): boolean

  • Returns true if two points are within a specified range. Provide a point for the range to set different x/y range, or pass a number to use the same range for both axis.

    example
    withinRange({x:100,y:100}, {x:101, y:101}, 1); // True
    withinRange({x:100,y:100}, {x:105, y:101}, {x:5, y:1}); // True 
    withinRange({x:100,y:100}, {x:105, y:105}, {x:5, y:1}); // False - y axis too far

    Parameters

    • a: Point
    • b: Point
    • maxRange: number | Point

    Returns boolean
