addValueByPath

  • addValueByPath(value, path, pathOpts?): void

  • Adds a value by a string path. Automatically generates intermediate nodes.

    const rootValue = {}
    const root = treeNodeMutable(rootValue, 'pc');
    root.addValueByPath({x:'c'},  'c');
    root.addValueByPath({x:'admin'}, 'c.users.admin');

    Creates the structure:

    pc         {}
    + c        {x: 'c' }
     + users   undefined
      + admin  {x: 'admin'}

    Parameters

    • value: V
    • path: string
    • pathOpts: PathOpts = {}

    Returns void
