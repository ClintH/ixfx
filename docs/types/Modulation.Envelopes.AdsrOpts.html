Type alias AdsrOpts

AdsrOpts: {
    attackBend?: number;
    decayBend?: number;
    initialLevel?: number;
    peakLevel?: number;
    releaseBend?: number;
    releaseLevel?: number;
    retrigger?: boolean;
    sustainLevel?: number;
}

Options for the ADSR envelope.

Use defaultAdsrOpts to get an initial default:

Type declaration

  • Optional Readonly attackBend?: number

    Attack bezier 'bend'. Bend from -1 to 1. 0 for a straight line

  • Optional Readonly decayBend?: number

    Decay bezier 'bend'. Bend from -1 to 1. 0 for a straight line

  • Optional Readonly initialLevel?: number

    Starting level (usually 0)

  • Optional Readonly peakLevel?: number

    Peak level (maximum of attack stage)

  • Optional Readonly releaseBend?: number

    Release bezier 'bend'. Bend from -1 to 1. 0 for a straight line

  • Optional Readonly releaseLevel?: number

    Release level, when envelope is done (usually 0)

  • Optional Readonly retrigger?: boolean

    When false, envelope starts from it's current level when being triggered. True by default.

  • Optional Readonly sustainLevel?: number

    Sustain level. Only valid if trigger and hold happens

Example

let env = adsr({
 ...defaultAdsrOpts(),
 attackDuration: 2000,
 releaseDuration: 5000,
 sustainLevel: 1,
 retrigger: false
});
